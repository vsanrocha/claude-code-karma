## Global Observations

### Theme Variables (globals.css)

| Variable | Light Mode Value | Dark Mode Value |
|----------|------------------|-----------------|
| `--background` | `0 0% 100%` | `224 71% 4%` |
| `--card` | `0 0% 100%` | `224 71% 4%` |
| `--primary` | `262 83% 58%` | `263 70% 50%` |
| `--muted-foreground` | `220 9% 46%` | `215 20% 65%` |

- `--background` and `--card` use identical values in dark mode
- Total CSS in globals.css: 57 lines
- No `--bg-secondary` or `--bg-tertiary` variables defined
- No dedicated elevation tier variables

### Typography Observed

| Element | Class Used | Computed Size |
|---------|------------|---------------|
| Page title | `text-3xl font-bold` | 30px, weight 700 |
| Stat card value | `text-2xl font-bold` | 24px, weight 700 |

### Navigation

- Top nav contains: Logo, "Projects" link, "Analytics" link
- No command palette (Cmd+K) present
- No skip-to-content link at page start
- Breadcrumb pattern: "← Back to projects" / "← Back to project"

## Component Implementation Observations

### StatsCard Component (`stats-card.tsx`)

```typescript
interface StatsCardProps {
  title: string;
  value: string | number;
  description?: string;
  icon?: LucideIcon;
  trend?: { value: number; label: string; };
  className?: string;
}
```

- Trend prop exists but not used in any current implementation
- Icon rendered in 32x32 container with `bg-primary/10` background
- Value uses `text-2xl font-bold`

### SessionCard Component (`session-card.tsx`)

- Model name formatting function present: transforms `claude-3-5-sonnet-20241022` → `3.5 Sonnet (20241022)`
- Fallback for session name: `session.slug || session.uuid.slice(0, 8)`
- Truncates initial_prompt to 150 characters
- No sanitization of XML/HTML content in initial_prompt

### TimelineRail Component (`timeline-rail.tsx`)

- 987 lines total
- Filter categories: `"prompt" | "tool_call" | "subagent" | "todo_update"`
- Uses `useMemo` for counts calculation
- Uses `useCallback` for filter toggle
- Event config defines 6 event types with distinct colors
- Tool-specific icons mapped for 15 different tools

## Accessibility Observations

### Present

- `aria-pressed` on FilterChip buttons
- `aria-hidden="true"` on decorative icons
- `:focus-visible` ring styles on interactive elements
- `cursor-pointer` on clickable elements

### Absent

- No skip-to-main-content link
- No `aria-label` on chart containers
- No `role="img"` with alt text on data visualizations
- No live region (`aria-live`) for filter count updates
